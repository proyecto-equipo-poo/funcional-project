# Functional Project - Java POO

## Descripción

Este repositorio contiene un proyecto funcional desarrollado en Java utilizando Programación Orientada a Objetos (POO).

El proyecto fue realizado con fines académicos para demostrar la aplicación práctica de conceptos fundamentales de POO como:

- Clases y objetos
- Encapsulamiento
- Herencia
- Polimorfismo
- Constructores
- Interfaces
- Manejo de excepciones

---

## Objetivo

Desarrollar una aplicación funcional en Java aplicando buenas prácticas de Programación Orientada a Objetos.

---

## Tecnologías utilizadas

- Java
- IntelliJ IDEA
- Git
- GitHub

---

## Funcionalidades

- Creación de objetos
- Uso de clases y métodos
- Encapsulamiento mediante getters y setters
- Herencia entre clases
- Polimorfismo dinámico
- Manejo de excepciones
- Organización modular del código

---

## Estructura del proyecto

```plaintext
src/
├── main/
│   ├── clases
│   ├── modelos
│   ├── servicios
│   └── Main.java
```

---

## Ejecución del proyecto

1. Abrir el proyecto en IntelliJ IDEA.
2. Verificar que el JDK esté configurado.
3. Ejecutar la clase principal `Main.java`.
